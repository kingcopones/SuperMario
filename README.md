# Super Mario Characters 
An app that displays the Super Mario Characters and redirection to Google Search for details

## MVVM Design Pattern
- Model (UserModel)
- View (MainScreenView)
- View Model (UsersViewModel)

## Wrappers / Utils
- Shimmer Loading (ShimmerCardView)
- WebKit
